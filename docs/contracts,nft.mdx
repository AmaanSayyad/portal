This page showcases client usage for the Admin SDK

## Getting Started

The NFT interface allows anyone to fetch the public metadata about any of the on-chain NFTs, and it allows users to check their NFT ownership and transfer their NFTs to others.

To initialize the SDKs NFT interface, you can pass in the relevant smart contract address by following the code below.

```js
const sdk = new ThirdwebSDK(...)

// Get nft contract address from admin ui
const address = "0x...";

// Initialize nft SDK by passing in contract address
const nftModule = sdk.getNFTModule(address);
```

## Interfaces

```ts
interface NFTMetadata {
  id: string;
  uri: string;
  name?: string; // optional
  description?: string; // optional
  image?: string; // optional
  properties?: Record<string, JSONValue>; // optional
}
```

## Functions

### get

`get(tokenId: string): Promise<NFTMetadata>`

Gets the metadata of a specific on-chain NFT including its name, description, and image.

```js
// the tokenId of the nft that you want to fetch
const tokenId = "42";

// returns nft metadata of requested nft
const metadata = await nftModule.get(tokenId);

// nft metadata is now directly available
const { name, description, image } = metadata;
```

### getAll

`getAll(): Promise<NFTMetadata[]>`

Get the metadata of all on-chain NFTs.

```js
// returns list of nft metadata of all on-chain nfts
const nfts = await nftModule.getAll();

nfts.map((metadata) => {
  // nft metadata is directly available off each item
  const { name, description, image } = metadata;
});
```

### balance

`balance(tokenId: string): Promise<BigNumber>`

Get the NFT balance of the user currently connected to the SDK for a specific NFT.

```js
// the tokenId of the nft that you want to fetch
const tokenId = "42";
// get the nft balance of the account currently connected
const balance = await nftModule.balance(tokenId);
```

### balanceOf

`balanceOf(address: string, tokenId: string): Promise<BigNumber>`

Get the NFT balance of a user specified by their public address for a specific NFT.

```js
// the tokenId of the nft that you want to fetch
const tokenId = "42";

// the address of the user you want to fetch the balance of
const address = "0x...";

// get the nft balance of the specified account
const balance = await nftModule.balanceOf(address, tokenId);
```

### transfer

`transfer(tokenId: string, to: string): Promise<void>`

Transfer a specific NFT from the user connected to the SDK to another user by specifying their address.

```js
// the tokenId of the nft that you want to send
const tokenId = "42";

// the address of the user you want to transfer the nft to
const to = "0x...";

// wait for transfer transaction
await nftModule.transfer(to, tokenId);
```
